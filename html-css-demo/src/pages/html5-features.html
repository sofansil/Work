New Semantic Elements

HTML5 introduced new semantic elements like <header>, <footer>, <article>, and <section> that help to structure the content better.

Audio and Video Support

HTML5 provides native support for audio and video embedding using the <audio> and <video> elements.

Canvas Element

The <canvas> element allows for dynamic, scriptable rendering of 2D shapes and bitmap images.

Geolocation API

HTML5 includes a Geolocation API that allows the user's location to be accessed with their permission.
